Privacy Policy

1. Introduction

steamengine respects your privacy. This Privacy Policy explains how information is collected, used, and protected when you use this application.

steamengine is a locally running application that operates on the user's own device and communicates with TikTok services only when authorized by the user.

2. Information We Collect

The application may process the following information:

  • TikTok account information provided through TikTok OAuth authentication.
  • Authentication tokens required to connect with TikTok APIs.
  • Video files and publishing information selected by the user.
  • Application settings required for normal operation.

3. How We Use Information

Information is used only to provide the application's functionality, including:

  • Authenticating the user's TikTok account.
  • Uploading and publishing videos requested by the user.
  • Managing the application's local settings.
  • Maintaining reliable operation.

4. Local Application Processing

AutoTikTok Publisher runs locally on the user's computer. The application does not operate a public cloud server that stores user content.

Any required information, such as application settings or authentication data, may be stored locally on the user's device.

5. TikTok API Usage

The application uses TikTok's authorized APIs to perform actions requested by the user. Access is granted only after user authentication and approval.

6. Information Sharing

We do not sell, rent, or trade user information.

Information is only transmitted to third-party services when necessary to provide the requested functionality, such as communicating with TikTok APIs.

7. Data Security

We take reasonable measures to protect information processed by the application. Users are responsible for maintaining the security of their own device and account.

8. Data Retention

Because the application runs locally, users control locally stored information. Users may remove stored application data by deleting the application files or local configuration data.

9. User Control

Users can revoke the application's access to their TikTok account at any time through TikTok account settings.

10. Children's Privacy

This application is not intended for children under the minimum age required by applicable laws and platform policies.
